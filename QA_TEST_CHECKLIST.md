# BDR Tournament Recorder - QA Test Checklist

> Phase 9: QA & 최적화 테스트 체크리스트

## 1. 오프라인 모드 테스트

### 1.1 기본 오프라인 기능
- [ ] 비행기 모드에서 앱 시작 가능
- [ ] 캐시된 대회/팀/선수 데이터 표시
- [ ] 새 경기 생성 가능 (오프라인)
- [ ] 경기 기록 저장 (로컬 DB)

### 1.2 데이터 지속성
- [ ] 앱 강제 종료 후 데이터 유지
- [ ] 기기 재시작 후 데이터 유지
- [ ] 진행 중 경기 복구 다이얼로그 표시
- [ ] "계속하기" 선택 시 정확한 상태 복원

### 1.3 네트워크 전환
- [ ] 온라인 → 오프라인 전환 시 알림 표시
- [ ] 오프라인 → 온라인 전환 시 자동 동기화 시도
- [ ] 네트워크 불안정 시 데이터 손실 없음

---

## 2. 경기 전체 플로우 테스트

### 2.1 경기 시작
- [ ] 대회 연결 (QR/토큰)
- [ ] 경기 선택 화면 진입
- [ ] 선발 선수 선택 (팀당 5명)
- [ ] 선발 선수 저장 및 경기 시작

### 2.2 쿼터 진행
- [ ] Q1 시작 → 타이머 동작
- [ ] 쿼터 종료 시 자동 점수 저장
- [ ] Q2 시작 → 팀 파울 리셋
- [ ] 하프타임 표시 (Q2 → Q3)
- [ ] Q3, Q4 정상 진행
- [ ] 연장전 진입 (동점 시)

### 2.3 점수 기록
- [ ] 2점 슛 성공/실패
- [ ] 3점 슛 성공/실패
- [ ] 자유투 성공/실패 (1/2/3구)
- [ ] 코트 위치 터치 → 슛차트 기록
- [ ] 점수 합계 = 2P×2 + 3P×3 + FT×1

### 2.4 액션 기록
- [ ] 어시스트 기록
- [ ] 리바운드 (공격/수비)
- [ ] 스틸 → 상대 턴오버 자동
- [ ] 블락 → 상대 슛 실패 자동
- [ ] 턴오버 기록
- [ ] 파울 기록 (개인/테크니컬)
- [ ] 5파울 퇴장 알림

### 2.5 선수 교체
- [ ] 교체 패널 열기
- [ ] 선수 선택 (IN/OUT)
- [ ] 교체 확인 및 저장
- [ ] 출전 시간 자동 계산

### 2.6 경기 종료
- [ ] 경기 종료 버튼
- [ ] 최종 통계 확인 화면
- [ ] 경기 저장 완료
- [ ] 동기화 시도 (온라인 시)

---

## 3. 동기화 테스트

### 3.1 정상 동기화
- [ ] 경기 종료 후 자동 동기화
- [ ] 수동 동기화 버튼 동작
- [ ] 동기화 성공 시 상태 업데이트

### 3.2 동기화 실패 처리
- [ ] 네트워크 오류 시 재시도
- [ ] 서버 오류 시 에러 메시지
- [ ] 미동기화 데이터 표시

### 3.3 충돌 해결
- [ ] 서버 데이터와 로컬 데이터 충돌 감지
- [ ] 충돌 해결 전략 적용 (로컬 우선/서버 우선)
- [ ] 충돌 로그 기록

---

## 4. 배터리 & 성능 테스트

### 4.1 배터리 모니터링
- [ ] 배터리 상태 아이콘 표시 (앱바)
- [ ] 20% 이하 경고 배너
- [ ] 10% 이하 위험 배너
- [ ] 충전 시 경고 해제

### 4.2 자동 저장
- [ ] 30초 간격 자동 저장
- [ ] 자동 저장 상태 아이콘 표시
- [ ] 앱 백그라운드 시 즉시 저장
- [ ] 앱 포어그라운드 복귀 시 재개

### 4.3 성능
- [ ] 경기당 300+ 플레이 기록 시 성능 유지
- [ ] 슛 기록 응답 시간 < 100ms
- [ ] 화면 전환 시 렉 없음
- [ ] 메모리 누수 없음 (장시간 사용)

---

## 5. 태블릿 펜 입력 테스트 (Galaxy Tab S9)

### 5.1 에어 커맨드
- [ ] 펜 호버 시 퀵 메뉴 표시
- [ ] 펜 떼기 시 메뉴 유지 (짧은 딜레이)
- [ ] 펜 멀리 이동 시 메뉴 닫힘
- [ ] 메뉴 항목 정확한 탭 인식

### 5.2 코트 터치
- [ ] 정확한 좌표 인식
- [ ] 슛 위치 시각적 표시
- [ ] 존(zone) 자동 계산
- [ ] 거리(distance) 자동 계산

### 5.3 일반 터치 대응
- [ ] 펜 없이 손가락으로 동일 기능
- [ ] 롱프레스로 메뉴 열기
- [ ] 멀티터치 충돌 없음

## 6. 에러 처리 & 복구 테스트

### 6.1 앱 크래시 복구
- [ ] 경기 중 앱 강제 종료
- [ ] 앱 재시작 시 복구 다이얼로그
- [ ] "계속하기" → 정확한 상태 복원
- [ ] "새로 시작" → 기존 경기 취소

### 6.2 데이터 무결성
- [ ] 선수 스탯 합계 = 팀 점수
- [ ] PlayByPlay 기록 완전성
- [ ] 쿼터별 점수 정확성
- [ ] 파울 카운트 정확성

### 6.3 에러 메시지
- [ ] 명확한 에러 메시지 표시
- [ ] 복구 가능한 액션 안내
- [ ] 로그 기록 (디버깅용)
