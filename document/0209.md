## 요구사항 
# 기본 요구사항
1. NBA.com의 boxscore가 모티브가 되면 됨. 

# 디자인 요구사항
1. 상단 점수
2. 우측 로그 / 농구 경기 기록 관련 된 모든 것. 
3. 중앙 .농구경기장을 가로로 길게 두고 "백넘버 중심으 동그란 아이콘으로 선수 구분이 가능하도록 함. 
4. ㄴ선수 아이콘을 누르면 슛/리바/블락스틸 등의 스탯들이 에어커멘드 효과로 등록할수 있게 하면됨. 
5. 하단에는 후보선수들의 백넘버가 있어서, 드래그앤 드랍으로 선수교체가 가능하도록 만들면 됨. 


# 기능요구사항
1. bdr_platform 의 대회 관련 데이터베이스와 연결이 되어야 해. 
2. 연결은 api 로하고 
3. 어플은 일단 태블릯 가로형으로 진행
4. 어플이 온라인 상태가 오류가 발생할 수 있으니 시작전 (당일)대회 데이터는 전부 업데이트 받은 상태에서 오프라인은로도 기록이 가능하도록 할꺼야. 
5. 디바이스내 디비에 경기 내용(하루동안 다수의 경기가 저장될수 있음) 이 저장됨
6. 저장된 데이터를 다시 온라인 상태에서 bdr_platform과 싱크를 맞출수 있도록 수동으로 진행하는 기능이 필요. 
7. 꼭 마지막에 싱크가 아니더라도, (온라인 상태가 유지된다면) 경기가 끝날때마다 싱크를 맞출수도 있음.
